# README

## Description

해당 프로젝트는 SW 개발자들을 위한 모바일 웹 사이트를 구현합니다. 다양한 사이트에 퍼져있는 SW 공모전 정보들을 모아 사용자에게 추천합니다. 사이트를 통해 공모전과 스터디의 팀원을 구할 수 있고, 정보에서 사용자의 팔로우에 따른 뉴스피드를 볼 수 있도록 편의를 제공하고 있습니다.
